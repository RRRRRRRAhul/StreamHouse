# 🎬 StreamHouse - Video Streaming Platform

A full-stack video streaming application built using **React.js** and **Django REST Framework (DRF)**.  
This platform allows users to upload, stream, and manage videos efficiently with authentication support.

---

## 🚀 Features

- 🔐 User Authentication (JWT based)
- 📤 Video Upload System
- 🎥 Video Streaming using `StreamingHttpResponse`
- ⏩ Supports Range Requests (Partial Content - 206)
- 📂 Category Management
- 📱 Responsive UI with React + Bootstrap
- 🔄 State Management using Redux Toolkit
- 🌐 RESTful API Integration

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Redux Toolkit
- Bootstrap
- Axios / Fetch API

### Backend
- Django
- Django REST Framework (DRF)
- JWT Authentication
- StreamingHttpResponse (for video streaming)

### Tools & Others
- Git & GitHub
- Postman (API Testing)
